## Ollama Cloud

This service uses Ollama Cloud with your own API key.

1. **Get an API key** from [ollama.com](https://ollama.com)
2. **Enter your API key** when enrolling in the service

## Best Practices

1. **Rate Limiting**: Implement exponential backoff for rate limit handling
2. **Error Handling**: Always handle API errors gracefully
3. **Security**: Never expose any API keys in client-side code
